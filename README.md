`NODE_ENV`

`PROJECT_JIRA`

`LOGGER_LEVEL`

`LOGGER_CLOUD_ENV`

`LOGGER_NAME`

## Run Locally

Clone the project

```bash
  git clone https://github.com/davivienda-colombia/davi-SYMPHONY_NAME_PROJECT
```

Go to the project directory

```bash
  cd davi-SYMPHONY_NAME_PROJECT
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start:dev
```

## Running Tests
